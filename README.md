# URL Shortener

This repository contains the Node.js API.

## Getting Started

### Clone the repo

$ git clone https://AndryK1980@bitbucket.org/AndryK1980/url-shortener.git
$ cd url-shortener

### Install dependencies

$ npm install     

### Start db

(Terminal 1) $ <your path fo DB> mongod
(Terminal 2) $ <your path fo DB> mongo

## Development server

$nodemon server.js
